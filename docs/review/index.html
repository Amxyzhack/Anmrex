Security Architecture Review

How Anmrex Protects User Funds

Anmrex employs a multi-layered, in-depth defense system across fund custody, account access, and real-time transaction monitoring. Whether you use the web platform or the Anmrex app, every interaction is protected by the same institutional-grade stack.

🥶

90%+ Cold Wallet Storage

The vast majority of user funds are held in air-gapped cold wallets, with hot wallets retaining only the liquidity needed for daily operations, minimizing the external attack surface.

🔑

MPC + Multi-Signature

Multi-party computation and multi-signature protocols require joint authorization from multiple independent nodes before any fund transfer can be executed — eliminating single-point key risks.

🛡️

HSM Key Sharding

Hardware Security Modules protect cryptographic key generation, storage, and usage across the full lifecycle, ensuring keys are never exposed in plaintext.

🚫

Zero-Trust Architecture

Every access request is verified and dynamically authorized in real time. Multi-factor authentication combines dynamic OTPs, device fingerprinting, and biometric verification.

🤖

AI-Driven Risk Engine

Machine learning models monitor all transactions in real time, detecting anomalies, potential money laundering patterns, and market manipulation with automatic intervention capabilities.

⛓️

On-Chain Compliance Monitoring

Address tagging and fund flow analysis — powered by Chainalysis and Elliptic — continuously track suspicious transaction paths across platforms.
