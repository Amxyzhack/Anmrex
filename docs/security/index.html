Architecture Overview

A Multi-Layered Defense System

Anmrex employs a defense-in-depth approach: no single layer is relied upon alone. Each layer independently limits risk, so that a failure in one does not compromise the whole.

L1
🥶

Cold Storage — Fund Custody Layer

Over 90% of all user assets are held in air-gapped cold wallets, completely disconnected from the internet. Hot wallets retain only the minimum liquidity required for daily operations, dramatically reducing the external attack surface.

L2
🔑

MPC + Multi-Signature — Authorization Layer

Every fund transfer requires joint authorization from multiple independent nodes via multi-party computation and multi-signature protocols. No single person or system can unilaterally move user funds — eliminating single-point key risk entirely.

L3
🛡️

HSM Key Sharding — Key Protection Layer

Hardware Security Modules protect cryptographic keys throughout their full lifecycle: generation, storage, and usage. Key sharding ensures that no complete private key ever exists in a single location or system.

L4
🚫

Zero-Trust Architecture — Account Access Layer

Every access request is treated as potentially hostile and requires real-time verification and dynamic authorization. Multi-factor authentication combines dynamic OTPs, device fingerprinting, and biometric identification to ensure only legitimate users can access accounts.

L5
🤖

AI Risk Engine — Transaction Monitoring Layer

Machine learning models trained on large-scale transaction data monitor all platform activity in real time. The system identifies anomalous patterns, potential money laundering behavior, and market manipulation — triggering automated responses including freezing, isolation, or account limiting, with simultaneous escalation to manual review.

L6
⛓️

On-Chain Compliance — Cross-Platform Risk Layer

Powered by partnerships with Chainalysis and Elliptic, Anmrex continuously monitors the on-chain origin and destination of funds. Address tagging and fund flow analysis prevent cross-platform risk proliferation and identify high-risk wallet interactions before they affect the platform.

Account Protection

Zero-Trust Account Security

Anmrex treats every access request as potentially hostile — verification is continuous, not one-time.

🔐

Multi-Factor Authentication

Three independent verification factors: dynamic one-time passwords, device fingerprinting, and biometric identification. All three must align for account access to be granted.

📱

Device Fingerprinting

Every device used to access the platform is registered and continuously verified. Unrecognized device access triggers immediate secondary authentication and alert escalation.

⚡

Anomaly Interception

The anomaly detection engine monitors all account activity in real time. High-frequency requests, privilege escalation attempts, and unusual geographic access are intercepted automatically.

✍️

Critical Operation Verification

Withdrawal requests, security setting changes, and API key management all require secondary verification — preventing unauthorized use even on a compromised session.

🌐

Dynamic Authorization

Authorization is not static or session-based. Every sensitive operation triggers its own real-time authorization check, independent of prior session state.

📋

Full Audit Trail

Every account action is logged with timestamp, device, and IP information. The audit trail is immutable and available for review in compliance-related investigations.

Business Continuity

What Happens in an Emergency?

Anmrex maintains comprehensive business continuity and emergency response plans — tested at least annually.

01

Multi-Region Disaster Recovery

Disaster recovery systems are deployed across multiple global locations. Hot and cold backup mechanisms ensure critical services remain operational during single-point failures or large-scale attacks.

02

Cold Wallet Emergency Protocol

Cold wallet activation follows strict institutionalized procedures, enabling rapid fund migration in the shortest possible time when necessary, with full chain-of-custody documentation.

03

Annual Full-Scale Drills

At minimum annually, Anmrex conducts comprehensive drills: disaster recovery switching tests, penetration attack simulations, and risk control system stress tests — with documented results.

04

Crisis Communication Protocol

A pre-established crisis communication framework ensures rapid, accurate information delivery to users, partners, and regulators during emergencies — preventing secondary risk from misinformation.
